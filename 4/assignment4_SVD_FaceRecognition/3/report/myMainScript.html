Interpretation

There were a variety of heuristics that we tried out. One of them was comparing the minimum squared difference corresponding to the prediction, and then checking if that distance is below a particular threshold (or equivalently the inverse being above a certain threshold). We also tried a heuristic of ratio of min. squared distance to max. squared distance, with the intuition being that a robust classifier will give a very low distance for the correct class, and high distances for other classes, and for an image not in the database, it should give approximately similar distances for all classes. We tried the method, and it was a little less better performing than the direct thresholding method.
We used the training set of 32*8 images, and treated the (32*4 + 8*10) remaining images as a validation set (which was used to tune the value of threshold). We have given 2 images (in the images directory) which shows the values of inverse of minimum squared distance, and the ratio of min. and max. distances (inverted for a probabilistic interpretation -- less squared distance is a better indication of higher "probability" or "confidence" factor). The blue curve is for the images from the set of (32*8) images, and the red curve is for the set of (8*10) images.

clc;
clear;
imgPath = '../../images/att_faces/';
[training_set, testing_set] = getImages(imgPath);
newImages = getOtherImages(imgPath);
threshold = 0.023;

% training here
[x_bar, X, V] = eigenfaces(training_set);
k = 23;
V_hat = V(:, 1:k);
alpha = V_hat'*X;
alpha_size = size(alpha,2);

z = testing_set-repmat(x_bar, [1, size(testing_set,2)]);
b = V_hat'*z;

neg_z = newImages - repmat(x_bar, [1, size(newImages,2)]);
neg_b = V_hat'*neg_z;

% pos = zeros(size(b, 2), 1);
maxp = zeros(size(b, 2), 1);

false_negs = 0;
for i=1:size(b, 2),
    dist = sum((alpha - repmat(b(:,i),[1,alpha_size])).^2, 1);
    prob = 1./dist;
%     prob = sort(prob/sum(prob), 'descend');
%     ratio = prob(1) + prob(2);
%     pos(i) = ratio;
%     maxp(i) = prob(1);
    maxp(i) = max(prob);
    if(maxp(i) < threshold)
        false_negs = false_negs + 1;
    end
end

% pos = sort(pos);
% maxp = sort(maxp);

% neg_pos = zeros(size(neg_b, 2), 1);
maxnp = zeros(size(neg_b, 2), 1);


false_pos = 0;
for i=1:size(neg_b, 2),
	dist = sum((alpha - repmat(neg_b(:,i),[1,alpha_size])).^2, 1);
    prob = 1./dist;
%     prob = sort(prob/sum(prob), 'descend');
%     ratio = prob(1) + prob(2);
%     neg_pos(i) = ratio;
%     maxnp(i) = prob(1);
    maxnp(i) = max(prob);
    if(maxnp(i)>=threshold)
        false_pos = false_pos + 1;
    end
end

fprintf('Number of false positives is %d\n', false_pos);
fprintf('Number of false negatives is %d\n', false_negs);
% neg_pos = sort(neg_pos, 'descend');
% maxnp = sort(maxnp, 'descend');

toc;
